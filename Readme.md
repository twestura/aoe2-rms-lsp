# Aoe2 RMS LSP

A [Language Server Protocol](https://microsoft.github.io/language-server-protocol/) (LSP) server for Age of Empires II Random Map Scripts (`.rms`).

## Features
 
### Hover Documentation
 
Hovering over any RMS token displays documentation from Zetnus's [Definitive Random Map Scripting Guide](https://docs.google.com/document/d/1jnhZXoeL9mkRUJxcGlKnO98fIwFKStP_OBozpr0CHXo).
