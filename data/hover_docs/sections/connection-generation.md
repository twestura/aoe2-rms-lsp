<CONNECTION_GENERATION>

Create terrain paths between lands.

- Paths are created between the origins of lands.
- A Dijkstra-like shortest path algorithm is used to connect land origins.
- Connections are processed in the order listed in this setions.
- If a connection between two lands is not possible, that connection is not produced.
