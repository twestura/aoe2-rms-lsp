```rms
includeXS FILEPATH
```

---

Includes an `.xs` file.

**Arguments**:
- `FILEPATH`: Path to the file to include.
  - Files must use the `xs` extension.
  - Filepaths may **not** be surrounded with double quotation marks.
  - The filepath is relative to the xs folder.

Included files are transferred in a multiplayer lobby, but are not included with recorded games.

**Example**: See [XS Scripting for Beginners](https://ugc.aoe2.rocks/general/xs/).
