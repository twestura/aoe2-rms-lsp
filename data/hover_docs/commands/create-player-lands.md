  /* Attributes */

---

Creates a starting land for every player.

- May be used more than once to give each player multiple starting lands.
- Players without any land cannot receive starting units or resources, and will start at a random location with only a Town Center, Villagers, and a Scout.
- Using [direct_placement](https://docs.google.com/document/d/1jnhZXoeL9mkRUJxcGlKnO98fIwFKStP_OBozpr0CHXo/edit?tab=t.0#heading=h.jbnjt99zhiqx) with `create_player_lands` disables circular land positioning.
