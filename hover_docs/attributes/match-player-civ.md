```rms
match_player_civ
```

---

Ensures that each faction receives their correct starting scout unit.
**Star Wars Galactic Battlegrounds only.**

- With the exception of this attribute, SWGB uses the same commands and attributes that are available in AoC.
- Usually used for the starting scout, but also works for other units such as unique units from the fortress.

**See more**: [match_player_civ](https://docs.google.com/document/d/1jnhZXoeL9mkRUJxcGlKnO98fIwFKStP_OBozpr0CHXo/edit?tab=t.0#heading=h.32nz9uxvp2kz)
