```rms
create_terrain TERRAIN {
  /* Attributes */
}
```

---

Paints a terrain on top of an existing terrain.

- Terrains are generated sequentially in the order they appear in the script.
- Terrain generation occurs after lands, elevation, and cliffs, but before connections.

**Arguments**:
- `TERRAIN`: A [terrain constant](https://docs.google.com/document/d/1jnhZXoeL9mkRUJxcGlKnO98fIwFKStP_OBozpr0CHXo/edit#heading=h.3bdjnf7tryyk) (default `GRASS`).

**Example**: Create a forest covering 10% of the map.
```rms
<TERRAIN_GENERATION>
create_terrain FOREST {
  base_terrain GRASS
  land_percent 10
}
```
