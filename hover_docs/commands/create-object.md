```rms
create_object OBJECT {
  /* Attributes */
}
```

---

Place the specified object.

- Objects are placed in the order they appear in the script.
- An object does not generate if there is no valid position available to place it.
- Only one object may be placed per tile under normal circumstances. See [force_placement](https://docs.google.com/document/d/1jnhZXoeL9mkRUJxcGlKnO98fIwFKStP_OBozpr0CHXo/edit?tab=t.0#heading=h.138scj5wa7v7) for an exception.
- Creating the `VILLAGER` object produces 3&nbsp;Villagers.
- Creating the `SCOUT` object prodcues
  - an Eagle Scout for the Aztecs and Maya.
  - a Champi Scout for the Inca, Mapuche, Muisca, and Tupi.
  - a Scout Cavalry for all other civilizations.
- [Walls](https://docs.google.com/document/d/1jnhZXoeL9mkRUJxcGlKnO98fIwFKStP_OBozpr0CHXo/edit#heading=h.q7o6xdvi0noo) have special behavior.

**Arguments**:
- `OBJECT`: An [object constant](https://docs.google.com/document/d/1jnhZXoeL9mkRUJxcGlKnO98fIwFKStP_OBozpr0CHXo/edit#heading=h.nvxriamulybh), or a group name defined with [create_object_group](https://docs.google.com/document/d/1jnhZXoeL9mkRUJxcGlKnO98fIwFKStP_OBozpr0CHXo/edit?tab=t.0#heading=h.nx4tooy9xi9).

}
