```rms
nomad_resources
```

---

Adds the cost of a Town Center (275&nbsp;Wood, 100&nbsp;Stone) to every player's starting resources.
- The actual cost is added per civilization, e.g. the Inca receive only 85&nbsp;Stone.

**Example**:

```rms
<PLAYER_SETUP>
nomad_resources
```
