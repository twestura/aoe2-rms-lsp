```rms
cliff_type CLIFF_TYPE
```

---

Specifies the color and style of cliffs.
DE only.

**Arguments**:
- `CLIFF_TYPE`: a [cliff type constant](https://docs.google.com/document/d/1jnhZXoeL9mkRUJxcGlKnO98fIwFKStP_OBozpr0CHXo/edit?tab=t.0#heading=h.uow2rvrpkup5) (default `CT_GRANITE`).
  - `CT_GRANITE`, `CT_DESERT`, `CT_SNOW`, `CT_MARBLE`, `CT_LIMESTONE`, `CT_TERRACE`

**Example**: Brown cliffs to match dirt and desert terrains.
```rms
<CLIFF_GENERATION>
cliff_type CT_DESERT
```
